## 配色方案

> 配色须使用 Hex 值，避免"蓝色"等模糊描述。如需支持深色模式，定义两套色值。

### 浅色主题（默认）

| 用途 | 色值 | 变量名 | 说明 |
| :--- | :--- | :--- | :--- |
| 主色 | `#[HEX]` | `--color-primary` | 品牌主色调 |
| 主色-悬停 | `#[HEX]` | `--color-primary-hover` | 交互反馈 |
| 辅助色 | `#[HEX]` | `--color-secondary` | 次要操作 |
| 成功 | `#[HEX]` | `--color-success` | 成功状态 |
| 警告 | `#[HEX]` | `--color-warning` | 警告状态 |
| 错误 | `#[HEX]` | `--color-error` | 错误状态 |
| 背景 | `#[HEX]` | `--color-bg` | 页面背景 |
| 文字-主 | `#[HEX]` | `--color-text-primary` | 主要文字 |
| 文字-次 | `#[HEX]` | `--color-text-secondary` | 辅助文字 |
| 边框 | `#[HEX]` | `--color-border` | 分割线/边框 |

### 深色主题（按需）

> 如不需要深色模式，删除此节即可。

| 用途 | 色值（深色） | 变量名 | 说明 |
| :--- | :--- | :--- | :--- |
| 背景 | `#[HEX]` | `--color-bg` | 深色页面背景 |
| 表面 | `#[HEX]` | `--color-surface` | 深色卡片背景 |
| 文字-主 | `#[HEX]` | `--color-text-primary` | 深色模式主文字 |
| 边框 | `#[HEX]` | `--color-border` | 深色模式边框 |

## 组件规范

### 按钮

| 类型 | 样式 | 用途 |
| :--- | :--- | :--- |
| Primary | 主色填充、白色文字 | 主要操作 |
| Secondary | 边框、主色文字 | 次要操作 |
| Danger | 红色填充、白色文字 | 危险操作 |
| Ghost | 无边框、主色文字 | 轻量操作 |

尺寸：`sm`（32px高）/ `md`（40px高）/ `lg`（48px高）

### 表单输入

- 默认高度：40px | Focus 边框：`--color-primary` | 错误边框：`--color-error`
- 禁用态：opacity 0.5, cursor: not-allowed

## 交互状态定义

> 每个动态组件必须覆盖以下四种状态：

| 状态 | UI 表现 | 说明 |
| :--- | :--- | :--- |
| **Loading** | 骨架屏 / Spinner | 数据加载中 |
| **Empty** | 空状态插图 + 引导操作 | 无数据时 |
| **Success** | Toast 通知 | 操作成功 |
| **Error** | 内联错误信息 | 操作失败 |
